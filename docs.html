CodeIgniter 3.x

CodeIgniter 3.x 是框架的最新稳定版，仍在持续开发中。

《用户手册》包含入门介绍、教程、 一些"how to" 引导, 还有一些框架组件的参考文档。

CodeIgniter 3.x 遵循 MIT License.

Note: 《用户手册》定期重建于项目的 "develop" 分支，包含一些未正式发布功能的介绍， 在线版本《用户手册》对应当前下载框架的内容。

CodeIgniter 2.x

《用户手册》包含入门介绍、教程、 一些"how to" 引导, 还有一些框架组件的参考文档。

CodeIgniter 2.x 遵循 EllisLab open source license.
