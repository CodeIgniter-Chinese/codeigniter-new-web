CodeIgniter 开发者社区

CodeIgniter 是一个开发者社区的开源项目，
有不同的社区版块供开发者收集和交换意见。

论坛

我们的论坛使用 MyBB 搭建，有4个主要部分:

  • 常规事务 (新闻 & 讨论， 休息室，社区事务)
  • 使用 CodeIgniter (选择 CodeIgniter， 安装 & 配置，模型-视图-控制器，类库 & 辅助函数， 最佳实践)
  • 版本演进 (CodeIgniter 3.x，CodeIgniter 2.x，问题反馈)
  • 其他资源 (第三方插件，招聘和工作，教程发布，项目展示)

在论坛里你可以寻求帮助或讨论在框架使用中遇到的问题。

旧版本 CodeIgniter 论坛保留在EllisLab的服务器上。EllisLab已将论坛内容提供给我们，这些内容已经合并到新版论坛，所以内容是一致的。

浏览论坛

IRC.

CodeIgniter 开通了网络中继聊天室，名称 #codeigniter。
你可以使用你常用的 IRC 客户端进入聊天室，或使用 webchat。我们网站的 IRC 页面已集成 webchat 框架.

IRC 聊天室经常会有超过80人在线，但大部分人分布在不同时区，他们一直保持登录状态，以便下次在线时能够看到所有的聊天内容。 如果你是 IRC 新用户，请先阅读 IRC入门介绍 。

使用 I.R.C.

Github

所有开发行为都记录在 Github。 更多细节请查看贡献页面。

你可以在 Github 问题页面提交错误报告，或者提交请求增强框架功能或修复框架错误。我们不在 Github 上提供技框架相关的术支持和帮助... 请使用论坛代替。

我们也使用Github 问题页面跟进开发计划和采纳增强功能意见，会绑定到特定的版本中。
